01
Definition · Principle

CT의 개념과 정의

CT란? Computed Tomography

  • 기존의 X선 영상법 + 컴퓨터의 결합에 의한 영상화 방법
  • 인체 각 조직의 감약계수(μ) 차이를 이용
  • 횡단면(axial) 영상을 기본으로 다방면 재구성 가능
📚 출처:1장_CT의_기초이론_복사본.pdf, p.4

한계 1일반 X선 영상

  • 3차원 구조물을 평면에 투영 → 구조 중첩현상
  • 원자번호/두께가 다른 물질의 감약 표현에 한계

한계 2단층촬영(Tomography)

  • X선관-필름 등속 이동으로 목적 단층만 명료화
  • 목적부위에도 흐림현상(blurring) 발생
  • 산란선으로 인한 대조도 저하
📚 출처:1장_CT의_기초이론_복사본.pdf, p.5–6

Characteristics · Pros & Cons

CT의 특성 · 장점 · 제한점

CT의 7가지 특성

  • 고관전압 이용
  • 우수한 해상능
  • 우수한 조직 감약계수 분해능 = 대조도 분해능
  • 얇은 슬라이스 단면 영상 획득
  • 해부학적 구조 + 기능적 정보 제공
  • 원하는 방향으로 재구성 가능
  • 3차원 영상화
📚 출처:1장_CT의_기초이론_복사본.pdf, p.7

고관전압 기법을 적용하는 이유 (★빈출)

  1. 피사체 투과력 증가 → 검출기 도달 X선량 증가 → 검출효율 극대화
  2. 관용도(latitude) 넓은 영상 획득
  3. 조직의 감약계수 차 감소
  4. 선속경화(beam hardening)로 인한 아티팩트 감소
  5. 감약계수차가 큰 조직에서 발생하는 아티팩트 감소
📚 출처:1장_CT의_기초이론_복사본.pdf, p.8

PROSCT의 장점

  • 스캔시간 단축 → 1회 조영제 주입으로 multi-phase 검사 가능
  • ROI/픽셀 분석 → 정량적 진단
  • 비침습적 혈관조영술
  • 3D 영상화, perfusion, 가상내시경
  • 미세 병변 감별 용이

CONSCT의 제한점

  • 방사선 피폭
  • 고가의 검사비용
  • 조영제 부작용
  • 감약계수가 큰 물질에서 아티팩트 발생
📚 출처:1장_CT의_기초이론_복사본.pdf, p.9–10
03
Scanning Geometry

세대별 CT 장치

감약된 X선을 검출기에서 획득하는 방식에 따라 세대별로 구분. 빔 형태, 검출기 수, 운동 방식이 핵심 비교 포인트.

1세대1ST GENERATION

빔: Parallel beam (연필형)

검출기: 1–2개

운동: 회전 + 직선

용도: 두부 전용

2세대2ND GENERATION

빔: Fan beam (좁은 부채형)

검출기: 수십 개

운동: 회전 + 직선

특징: Bowtie filter 도입

3세대3RD GENERATION

빔: Rotating Fan beam (넓은 부채형)

검출기: 수백 개

운동: 회전 + 회전

특징: Ring artifact 발생

4세대4TH GENERATION

빔: Rotating/Nutating Fan beam

검출기: 수천 개

운동: 회전 + 고정

5세대5TH GENERATION

빔: Electron beam (EBCT)

검출기: 1,296개

운동: 고정 + 고정

특징: 전자총 사용

HelicalHELICAL / SPIRAL

원리: 나선형 스캔

핵심: Slip ring + brush

장점: 외부 고압케이블 제거 → 연속회전

X선관 회전 중 테이블 동시 이동

세대별 핵심 암기표

  1. 검출기 수 증가 순서: 1–2개 → 수십 → 수백 → 수천 → 1,296개(EBCT)
  2. Ring artifact = 3세대의 대표 결점 (회전/회전 운동)
  3. Bowtie filter 최초 도입 = 2세대
  4. Slip ring 기술 = Helical CT의 핵심 (연속회전 가능케 함)
  5. 5세대 EBCT = X선관과 검출기 모두 고정 (전자선 편향으로 영상 획득)
📚 출처:1장_CT의_기초이론_복사본.pdf, p.11–18
04
Imaging Chain

CT 영상화의 4단계

단계과정주요 내용
① X선 발생 필라멘트 가열 → 열전자 방출 → 회전양극 타깃 충돌 → X선 발생 (일반 X선과 동일)
② 감약 데이터 획득 Detector array + DAS(Data Acquisition System) → 아날로그 신호 → 증폭 → ADC로 디지털 변환
③ 데이터 처리 전처리 → 재구성 → Convolution(필터링) → 후처리
④ 영상 표시/저장 모니터 표시 또는 하드카피

③ 데이터 처리의 세부 단계

  • 전처리(Pre-processing): 측정값(measurement data)을 raw data로 변환 · 데이터 교정
  • 영상 재구성: Scan data → CT number로 변환 (수치화된 디지털 영상)
  • Convolution: 검사 목적/해부학적 특성에 맞게 필터링 — Standard · Bone · Soft tissue 알고리즘
  • 후처리: 부가적 영상처리
📚 출처:1장_CT의_기초이론_복사본.pdf, p.19–28

영상 표시의 한계와 Gray Scale

  • CT 시스템: 약 4,000개의 HU 존재
  • 모니터: 256개 음영만 표시 가능
  • → 이를 극복하기 위해 Gray scale + Windowing 기법 사용
📚 출처:1장_CT의_기초이론_복사본.pdf, p.29
05
Attenuation Physics

감약 · 선감약계수 · 반가층

감약(Attenuation) Attenuation

X선이 물질을 투과할 때 흡수되거나 산란되어 X선 강도가 낮아지는 현상.

X선 강도 = 광자수(Quantity, 양) × 광자 에너지(Quality, 질)

감약에 영향을 미치는 4인자:

  • X선 광자 에너지
  • 조직의 밀도
  • 조직의 원자번호
  • 물질의 단위중량당 전자 수
📚 출처:1장_CT의_기초이론_복사본.pdf, p.32–33

MONO단색방사선

  • 균등방사선 (homogeneous)
  • 광자수가 균일하게 감소
  • 최초 에너지 = 투과 후 에너지
  • 감약공식 성립
I = I₀ e−μx

POLY다색방사선

  • 불균등방사선 (heterogeneous)
  • 광자수가 불균일하게 감소
  • 투과 후 에너지 > 최초 에너지 (평균)
  • 감약공식 성립 안됨 → 선속경화

선속경화 현상 Beam Hardening

  • X선 진행 시 낮은 에너지 대역의 X선이 높은 에너지 대역보다 상대적으로 더 많이 감쇠
  • 결과: X선의 평균 에너지가 증가하는 현상
📚 출처:1장_CT의_기초이론_복사본.pdf, p.34–36

선감약계수(μ) Linear Attenuation Coefficient

  • 일정한 두께의 물질을 투과할 때 감소된 광자의 비율
  • 단색방사선에만 적용
  • 단위: cm⁻¹
N = N₀ · e−(μ₁+μ₂+⋯+μₙ)x

Hounsfield Unit

CT number (HU)

CT number의 정의 Hounsfield Unit

  • CT 영상을 구성하는 단위
  • 컴퓨터가 인체조직의 Matrix 수만큼 선감약계수를 측정·계산
  • 물을 기준(0)으로 하여 상대치로 나타낸 값
CT Number = k × (μₜ − μw) / μw
μₜ = 조직의 선감약계수 μw = 물의 선감약계수 k = 확대정수 (contrast factor / scaling factor) — 일반적으로 1,000
📚 출처:1장_CT의_기초이론_복사본.pdf, p.42, 44

기준 HU 값 (암기 필수)

공기 = −1,000 물 = 0 (기준) 뼈 · 조영제 = +1,000 금속 = +2,000 ~ +4,000

Gray scale: HU값이 클수록 밝은 음영, 작을수록 어둡게 표시.

CT number 영향 인자 (중요도 순)

  1. 선속 에너지 — 가장 큰 영향
  2. 피사체 원자번호
  3. 피사체 조직 밀도
  4. 조영 증강도

[기출 예제] 어떤 조직의 선감약계수가 0.247일 때 CT number 값은?
(단, 물 선감약계수 0.190, 뼈 감약계수 0.380)
STEP 1CT No = 1000 × (0.247 − 0.190) / 0.190
STEP 2= 1000 × (0.057 / 0.190)
STEP 3= 5,700 / 19
정답= 300 HU

Image Composition

픽셀 · 매트릭스 · 복셀 · ROI

슬라이스 두께와 간격 Slice Thickness & Interval

  • 두께: 인체 횡단면의 두께 (1 ~ 10 mm)
  • 간격: 한 슬라이스 중앙 → 다음 슬라이스 중앙까지의 거리 (1 ~ 수십 mm)
📚 출처:1장_CT의_기초이론_복사본.pdf, p.57

픽셀 Pixel (Picture Element)

  • 영상을 구성하는 최소 단위
  • 해부학적인 한 조각을 표현
  • 픽셀 크기가 작을수록 작은 조직 해부학 우수
  • 영상 모니터에서 gray scale의 한 점으로 표시
픽셀 크기 = scan FOV / (매트릭스 크기 × zoom factor) = display FOV / 매트릭스 크기
📚 출처:1장_CT의_기초이론_복사본.pdf, p.58

매트릭스 Matrix

  • 가로열 × 세로열의 2차원 픽셀 배열
  • 종류: 256×256, 320×320, 512×512
  • 256×256 매트릭스 = 총 65,536개 픽셀
📚 출처:1장_CT의_기초이론_복사본.pdf, p.59

ROI Region of Interest

  • 계측 목적으로 영상에 설정되는 2차원 영역
  • 구성: 원형, 타원형, 사각형, trace
  • ROI 설정 → CT HU, 노이즈 측정 가능
📚 출처:1장_CT의_기초이론_복사본.pdf, p.66

Data Types

Scan data vs Image data

RAWScan data (= Raw data)

  • 환자 투과 후 검출기에 포획된 감약 데이터
  • 또는 전처리 후 재구성 전 상태
  • 형태: Sinogram
  • 주파수 영역 데이터

IMGImage data

  • 공간 영역의 데이터 (눈으로 볼 수 있음)
  • 재구성 처리 후 digital로 변환
  • CT number로 표현된 데이터

데이터 크기 비교

  • Raw data 양 > Image data 양
  • → Raw data에서 FBP(Filtered Back Projection)를 통해 image data 재구성
📚 출처:1장_CT의_기초이론_복사본.pdf, p.60–62
용어정의
Construction X선관 고정, slit 형태 X선 노출, 테이블 이동하며 영상 획득 → Scout 영상처럼 스캔 즉시 영상 구성
Reconstruction 역투영법(Back projection) 등 재구성 기법으로 각 방향의 프로필 데이터 중첩 → Axial 영상 획득
Reformation 재구성된 Axial 영상을 이용하여 Coronal · Sagittal · Oblique 영상으로 변환
📚 출처:1장_CT의_기초이론_복사본.pdf, p.63–65

Windowing

Window Width & Level

CTCT 장치 Gray Scale

  • 12 bit
  • 총 4,096 단계
  • 범위: −1,024 ~ +3,071

MON모니터 Gray Scale

  • 8 bit
  • 총 256 단계
  • 범위: 0 ~ 255
📚 출처:1장_CT의_기초이론_복사본.pdf, p.70

Window Width (WW, 색소 폭)

  • CRT에 흑백으로 표시되는 CT No의 범위
  • 영상의 농도 범위 = Gray scale
  • WW가 높은 곳은 흰색, 낮은 곳은 검은색
  • CT 영상의 Contrast(대조도)를 결정
  • WW 작을수록 표시되는 CT No 폭이 좁고, 클수록 넓다

Window Level (WL, 색소 치)

  • Window 폭 범위의 중앙치
  • 명암(density) 준위 → 농도 변화
  • 직접 관련 인자: X선 흡수치(선감약계수)
  • WL 작을수록 WW 내 CT No 수가 낮고, 클수록 높아짐
📚 출처:1장_CT의_기초이론_복사본.pdf, p.71–73

WW↑Window Width ↑ (넓게)

  • Gray scale 증가
  • Contrast(대조도) 저하
  • 흑색·백색이 약해짐
  • Resolution(해상력) 저하
  • 비슷한 조직 식별능 저하

WW↓Window Width ↓ (좁게)

  • Gray scale 감소
  • Contrast(대조도) 증가
  • 흑색·백색이 강해짐
  • Resolution(해상력) 증가
  • 비슷한 조직 식별능 증가

WL↑Window Level ↑ (높이면)

  • 영상이 어두워짐
  • Bone처럼 X선 흡수도가 높은(μ 큰) 조직이 잘 관찰

WL↓Window Level ↓ (낮추면)

  • 영상이 밝아짐
  • 공기 · 지방처럼 X선 흡수도가 낮은(μ 작은) 조직이 잘 관찰
📚 출처:1장_CT의_기초이론_복사본.pdf, p.74–78

= 1000 × 0.095 / 0.190
