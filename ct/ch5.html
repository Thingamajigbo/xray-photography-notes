02
역투영법
Back Projection (BP)

역투영법 원리

Back Projection Principle

CT에서 영상 재구성이란 횡단면 내의 감약계수 분포를 한 방향에서 합계하여 얻은 1차원 분포(투영)를 다수의 방향에 대해 구하여, 이로부터 횡단면상을 복원하는 기법.

2×2 매트릭스로 이해하는 역투영 과정

원본: │ 0   2 │      → BP 복원 후: │ 0   2 │
        │ 1   3 │                           │ 1   3 │

절차 · (a) 수평방향 투영 → (b) 수직방향 투영 → (c) 두 투영의 합 → (d) 오른쪽 대각선 투영 합 → (e) 왼쪽 대각선 투영 합 → (f) Background 감산 → (g) 최소공배수로 나눔

결과 · 투영을 모든 방향에서 획득하고 역투영한 뒤 Background 감산 + 정규화 과정을 거치면 원본 매트릭스가 복원됨

역투영법(BP)의 결정적 문제점 Blurring Problem

단순 BP의 한계 — 흐림(Blurring)

Blurring Artifact in Simple BP

실제 물체는 좁은 영역에 집중된 projection 데이터로 나타나지만, 역투영을 하면 각 방향의 ray가 여러 픽셀에 함께 분배되어 누적되므로 결과 영상이 중앙이 높고 가장자리로 퍼지는 종모양(bell-shape)의 흐림 형태로 나타남.

  • 원인: 모든 방향에서 back projection한 값이 중첩되며 점 물체가 "star pattern" 형태로 번짐
  • 해결책 → Convolution filter를 추가한 FBP로 보정

역투영법(BP) 기출 포인트

  1. BP의 원리: 각 방향 ray를 거꾸로 픽셀면에 되돌려 합산
  2. BP의 핵심 문제: 흐림(blurring) 발생 → 영상이 중앙이 높은 종모양
  3. 점 물체 1개의 BP 결과: 여러 방향의 합성으로 star pattern이 나타남
  4. BP 단독으로는 부정확 → Convolution filter 보정이 필수 (즉 FBP로 발전)

출처: 5장 CT영상의 재구성기법, p.53~56

FBP (Filtered Back Projection)

Filtered Back Projection

CT 영상 재구성에 사용되는 대표적인 영상재구성 기법. 흐림을 제거하는 수학적 필터처리(Convolution)를 포함한다.

필터링의 수학적 의미

Mathematical Filtering Effect
  • 측정 데이터(a) → 흐림이 있는 봉우리 모양 (blur profile)
  • × 필터(b) → convolution kernel (중심 양수·주변 음수 형태)
  • = 필터링 후 결과(c) → 원래의 sharp한 프로파일로 복원

FBP 기출 포인트

  1. FBP = CT 영상 재구성에 사용되는 대표적인 기법
  2. 핵심: Convolution filter로 흐림(blur) 제거
  3. 장점: 타 기법 대비 처리속도 빠름 + 선명한 영상
  4. 단점: 노이즈 多 → FBP 후 노이즈 감소처리 필수
  5. FBP 5단계 순서 암기: 투영 → 전처리 → 필터 곱셈 → 역투영 → 합산

05
SSD — Shaded Surface Display
표면 음영 디스플레이

SSD (Shaded Surface Display)

Shaded Surface Display · Surface Rendering

가장 간단한 3차원 구성 방법. 관심 부분의 HU 범위와 일치하는 문턱값(threshold) 범위를 정하여, 그 범위 내의 HU를 지닌 복셀(voxel)들만 선택하여 표시하는 방법.

  • Opacity 그래프에서 선택 영역을 1, 외부는 0으로 이진 처리
  • 표면(surface)만 추출하여 음영 처리 → 입체감 표현
  • 주 적용: 뼈(골) 3D 재구성, 장기 표면 표시
  • 단점: 내부 정보 소실 · threshold에 매우 민감 · stair-step artifact 가능

출처: 5장 CT영상의 재구성기법, p.63~65

06
MIP — Maximum Intensity Projection
최대 강도 투영

MIP 원리

Maximum Intensity Projection

시선 방향을 따라 통과하는 모든 복셀 중 가장 높은 HU 값만을 선택해서 나타내는 기법.

예시: [1, 2, 6, 2, 3, 8, 6, 5] → Displayed = 8

원리 · Ray 경로상의 voxel 집합 중 max(HU)를 2D 영상 픽셀에 할당

적용 · 조영증강된 혈관(CT angiography), 석회화 병변 등 상대적 고농도 구조물 강조

출처: 5장 CT영상의 재구성기법, p.66~67

MinIP — Minimum Intensity Projection
최소 강도 투영

MinIP 원리

Minimum Intensity Projection

MIP의 반대 기법. Ray 경로상의 voxel 중 가장 낮은 HU 값만 선택해서 나타내는 기법.

예시: [8, 4, 5, 4, 1, 5, 3, 2, 5, 6] → Displayed = 1

원리 · Ray 경로상 voxel 집합 중 min(HU)를 영상 픽셀에 할당

적용 · 공기를 함유한 구조물 강조 → 폐의 기관지·세기관지, 담도 확장, 기종 병변 등

VRT — Volume Rendering Technique
용적 렌더링 기법 · 가장 우수한 3D 기법

VRT (Volume Rendering Technique)

Volume Rendering Technique

★ 가장 우수한 3차원 영상 기법 ★

  • 관심 영역별로 각각 다른 HU의 문턱값 범위를 여러 개 동시에 지정 가능
  • 각 범위에 색(color)을 부여하여 좀 더 입체적이고 사실적으로 표현
  • 선택 범위가 절대적이지 않고, 상하 영역의 일정 범위에 투과도(opacity)를 다르게 설정하여 재구성 영역 지정
  • 혈관(vessels, 빨강), 뼈(bone, 흰색), 수술 클립(clips, 파랑) 등 여러 조직을 한 영상에 동시 표현

VRT / 3D 기법 기출 포인트

  1. VRT = 가장 우수한 3차원 영상 기법 — 기출 최빈출
  2. VRT 특징: 여러 개의 HU threshold를 동시 지정 + 색 부여 + 투과도 조절
  3. VRT Definition 창(Preset, Opacity, Brightness, Color 조절 UI)을 보면 → 정답은 VRT
  4. 뼈·혈관 등을 한 화면에 다양한 색으로 표현한 실사적 3D 영상이면 VRT
  5. SSD는 표면만 음영 처리, VRT는 내부 구조까지 반투명 표현

Virtual Endoscopy 원리

Virtual Endoscopy

  3. 원리: 공기·조영제로 확장된 내강 표면을 내시경 시점으로 재구성

Multi-Planar Reformation

재구성된 얇은 슬라이스의 횡단면 영상을 이용하여, 다방면(multi-planar)으로 실시간 재구성하는 영상처리 기법.

  • 영상의 형태를 변형시키는 영상처리
  • 3D 기법이라기보다 2D 평면 재조합에 가까움

  • ⑤ 단순 sinogram 역연산
✓ 정답 · ③ 필터보정 역투영법 (FBP) FBP는 CT 재구성의 표준 기법으로, Convolution filter로 흐림을 제거한 뒤 back projection을 수행한다.
장점: 처리속도 빠름·선명한 영상 / 단점: 노이즈 多 → 노이즈 감소처리 필수.
▎ PROBLEM 02 빈출 · BASIC
다음 사진은 어떤 3차원 영상기법의 영상을 만들 때 이용되는가? (Preset, Opacity, Brightness, Color를 조절하는 VRT Definition 창)
  • ① SSD
  • ② VRT
  • ③ MIP
  • ④ MPR
✓ 정답 · ② VRT Opacity 조절, 색(Color) 부여, Brightness 조절이 가능한 UI는 Volume Rendering Technique의 정의 창이다. 여러 threshold를 동시에 지정하여 혈관·뼈·클립을 다른 색으로 동시 표현.
▎ PROBLEM 03 빈출 · BASIC
CT 영상을 이용한 3차원 가상내시경(Virtual Endoscopy) 기법에 적당하지 않는 부위는?
  • ① 위 (stomach)
  • ② 소장 (small intestine)
  • ③ 방광 (bladder)
  • ④ 기관지 (bronchus)
  • ⑤ 대장 (colon)
✓ 정답 · ② 소장 소장은 연동운동이 활발하고 내강이 좁으며 굴곡이 많아 공기·조영제로 균일하게 확장시키기 어렵다. 반면 위·대장·방광·기관지는 모두 내강을 확장시킬 수 있어 VE에 적합.
▎ PROBLEM 04 응용 · INTERMEDIATE
다음 중 MIP(Maximum Intensity Projection)의 특징으로 옳은 것은?
  • ① Ray 경로상 voxel의 평균값을 표시한다.
  • ② 가장 낮은 HU 값만 선택하여 표시한다.
  • ③ 가장 높은 HU 값만 선택하여 표시한다.
  • ④ 표면만 음영 처리하여 표시한다.
  • ⑤ 다중 threshold에 각각 색을 부여한다.
✓ 정답 · ③ 가장 높은 HU 값만 선택 MIP = Max(HU). 조영증강된 혈관(CT angiography), 석회화 병변 강조에 유용. ②는 MinIP, ④는 SSD, ⑤는 VRT.
▎ PROBLEM 05 개념 · BASIC
역투영법(BP)의 결정적 문제점은 무엇인가?
  • ① 계산량이 적어 정확도가 떨어진다.
  • ② 재구성된 영상에 흐림(blurring)이 발생한다.
  • ③ 환자 피폭선량이 급격히 증가한다.
  • ④ 양자화 오차를 보정하지 못한다.
  • ⑤ 프로젝션 수가 부족해진다.
✓ 정답 · ② 흐림(blurring) 발생 단순 BP는 모든 방향의 역투영이 중첩되어 점 물체가 중앙이 높은 종모양 / star pattern으로 흐려지는 한계가 있다. 이 흐림을 convolution filter로 제거한 것이 FBP.
▎ PROBLEM 06 통합 · APPLIED
필터보정 역투영법(FBP)의 단계 중 흐림 제거의 핵심이 되는 처리 단계는?
  • ① 투영 값 획득
  • ② 영상의 전처리
  • ③ Convolution filter 곱셈
  • ④ Back projection
  • ⑤ 합산 및 저장
✓ 정답 · ③ Convolution filter 곱셈 FBP의 수학적 핵심은 convolution filter (필터 커널) 적용이다. 중심 양수·주변 음수 형태의 kernel이 흐림 프로파일을 sharp profile로 복원시킨다.
Back projection은 최종 2D 영상 합성 단계.
▎ PROBLEM 07 개념 · BASIC
폐 기종·세기관지 확장 등을 평가하기 위해 공기를 함유한 저밀도 구조물을 강조해 보려 한다. 가장 적절한 3D 기법은?
  • ① SSD
  • ② VRT
  • ③ MIP
  • ④ MinIP
  • ⑤ MPR
✓ 정답 · ④ MinIP MinIP는 Ray 경로상의 voxel 중 최솟값(공기·저 HU)만을 표시하므로, 폐의 기관지·세기관지·기종 병변 평가에 최적. CT angiography는 MIP, 뼈 표면은 SSD, 혈관+뼈 동시 표현은 VRT.
▎ PROBLEM 08 빈출 · BASIC
다음 영상(척추·골반의 뼈가 다양한 음영과 색조로 입체적으로 재구성된 3D 영상)의 표현기법은?
  • ① SSD
  • ② VRT
  • ③ MIP
  • ④ MinIP
  • ⑤ MPR
✓ 정답 · ② VRT 여러 색상 + 실사적 입체감 + 반투명 표현 → VRT (Volume Rendering Technique). SSD는 단색 음영, MIP/MinIP는 2D 투영 영상, MPR은 평면 재구성.
▎ PROBLEM 09 응용 · INTERMEDIATE
MPR(Multi-Planar Reformation)에 해당하지 않는 평면은?
  • ① Coronal
  • ② Sagittal
  • ③ Oblique
  • ④ Curved
  • ⑤ Spectral
✓ 정답 · ⑤ Spectral MPR에는 Coronal, Sagittal, Oblique, Curved reformation이 포함된다. "Spectral"은 듀얼 에너지 CT의 스펙트럼 영상 용어로 MPR과는 무관.
▎ PROBLEM 10 통합 · ADVANCED
3D 재구성 기법 중 "가장 우수한 3차원 영상 기법"으로 평가되며, HU threshold 여러 개와 opacity·color를 동시 지정할 수 있는 것은?
  • ① SSD
  • ② MIP
  • ③ MinIP
  • ④ VRT
  • ⑤ Virtual Endoscopy
✓ 정답 · ④ VRT VRT (Volume Rendering Technique)는 관심 영역별로 각기 다른 HU 범위를 여러 개 지정하고, 각 범위에 색과 투과도(opacity)를 부여하여 가장 입체적·사실적으로 표현한다. 자료에서도 "가장 우수한 3차원 영상 기법"으로 명시됨.
