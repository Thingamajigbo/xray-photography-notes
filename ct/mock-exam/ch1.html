Q1. CT의 개념
기초 개념
CT 영상화의 기본 원리에 대한 설명으로 옳은 것은?
  • ①인체 조직의 자기공명 신호 차이를 이용한다
  • ②음파의 반사를 통해 단층 영상을 획득한다
  • ③인체 각 조직의 감약계수 차이를 이용하여 횡단면 영상을 얻는다
  • ④방사성동위원소의 붕괴를 검출하여 영상화한다
  • ⑤적외선의 투과도를 측정하여 재구성한다
해설
정답③

CT는 기존 X선 영상법과 컴퓨터의 결합에 의한 영상화 방법으로, 인체 각 조직의 감약계수(μ) 차이를 이용하여 횡단면(axial) 영상을 획득한다.

① MRI, ② 초음파, ④ 핵의학, ⑤ 적외선 영상의 원리이다.

Q2. CT의 특성
빈출 ★
CT에서 고관전압 기법을 적용하는 이유로 옳지 않은 것은?
  • ①피사체 투과력을 증가시켜 검출효율을 극대화한다
  • ②조직의 감약계수 차이를 증가시킨다
  • ③선속경화로 인한 아티팩트를 감소시킨다
  • ④관용도가 넓은 영상을 획득한다
  • ⑤감약계수차가 큰 조직에서 발생하는 아티팩트를 감소시킨다
해설
정답②

고관전압은 오히려 조직의 감약계수 차를 감소시킨다. 이는 선속경화 아티팩트를 줄이는 효과와 연결된다.

고관전압 적용 이유: 투과력 증가, 관용도 증가, 감약계수 차 감소, 선속경화 아티팩트 감소, 감약계수차 큰 조직의 아티팩트 감소.

Q3. 세대별 CT
빈출 ★
3세대 CT 장치의 특징으로 옳은 것은?
  • ①연필형 선속(parallel beam)을 사용한다
  • ②X선관과 검출기가 회전/직선 운동을 한다
  • ③X선관과 검출기가 회전/회전 운동을 하며 ring artifact가 발생한다
  • ④검출기가 고정되어 있고 X선관만 회전한다
  • ⑤전자총을 사용하여 X선관과 검출기가 모두 고정되어 있다
해설
정답③

3세대 CT는 Rotating Fan beam geometry(넓은 부채형 선속)를 사용하며, X선관과 검출기가 모두 회전(회전/회전 운동)한다. 검출기 수는 수백 개이며, 검출기 고장 시 ring artifact(원형 아티팩트)가 발생하는 것이 대표적 특징이다.

① 1세대, ② 1·2세대, ④ 4세대, ⑤ 5세대(EBCT)

Q4. Helical CT
빈출 ★
나선형(Helical) 스캔을 가능하게 한 핵심 기술 요소는?
  • ①Bowtie filter
  • ②Electron gun
  • ③Flat panel detector
  • ④Slip ring과 brush
  • ⑤Collimator array
해설
정답④

Helical CT는 Slip ring과 brush 구조를 이용하여 외부 고압케이블을 제거함으로써 X선관의 연속적인 회전이 가능해졌다. X선관이 회전하는 동안 환자 테이블도 동시에 이동하여 나선형 스캔이 이루어진다.

① Bowtie filter는 2세대부터 도입, ② Electron gun은 5세대 EBCT의 핵심이다.

📚 출처: 1장_CT의_기초이론_복사본.pdf, p.16–17
Q5. 선속경화
물리
선속경화(Beam Hardening) 현상에 대한 설명으로 옳은 것은?
  • ①단색방사선에서만 발생하는 현상이다
  • ②낮은 에너지 대역의 X선이 더 많이 감쇠되어 평균에너지가 증가하는 현상이다
  • ③X선 광자수가 균일하게 감소하는 현상이다
  • ④투과 후 X선 에너지가 감소하는 현상이다
  • ⑤감약공식 I = I₀e⁻ᵘˣ가 정확히 성립한다
해설
정답②

선속경화는 다색방사선(polychromatic)에서 발생하는 현상으로, X선이 진행하면서 낮은 에너지 대역의 X선이 높은 에너지 대역보다 상대적으로 더 많이 감쇠되어 X선의 평균 에너지가 증가하는 현상이다.

다색방사선에서는 감약공식 I = I₀e⁻ᵘˣ가 성립하지 않는다.

📚 출처: 1장_CT의_기초이론_복사본.pdf, p.34–36
Q6. CT number 계산
계산 ★★
어떤 조직의 선감약계수가 0.228이고, 물의 선감약계수가 0.190일 때 이 조직의 CT number는? (단, k=1,000)
CT No = k × (μₜ − μw) / μw
  • ①100 HU
  • ②150 HU
  • ③200 HU
  • ④250 HU
  • ⑤300 HU
해설
정답③

STEP 1. CT No = 1000 × (0.228 − 0.190) / 0.190

STEP 2. = 1000 × (0.038 / 0.190)

STEP 3. = 1000 × 0.2 = 200 HU

📚 출처: 1장_CT의_기초이론_복사본.pdf, p.42, 44–45
Q7. CT number 기준값
암기
다음 중 CT number(HU) 기준값이 잘못 짝지어진 것은?
  • ①물: −1,000 HU
  • ②공기: −1,000 HU
  • ③뼈(피질): 250 HU 이상
  • ④지방: −100 ± 10 HU
  • ⑤금속: +2,000 ~ +4,000 HU
해설
정답①

물은 CT number의 기준값인 0 HU이다. CT number는 물의 선감약계수를 기준으로 상대적으로 계산되기 때문이다.

공기 −1,000 / 물 0(기준) / 뼈·조영제 +1,000 / 금속 +2,000~+4,000

📚 출처: 1장_CT의_기초이론_복사본.pdf, p.30, 43
Q8. CT number 영향인자
빈출 ★
CT number에 가장 큰 영향을 미치는 인자는?
  • ①선속 에너지
  • ②피사체 원자번호
  • ③피사체 조직 밀도
  • ④조영 증강도
  • ⑤매트릭스 크기
해설
정답①

CT number 영향인자 중 선속 에너지(X-ray energy)가 가장 큰 영향을 미친다.

영향 순서: 선속에너지 > 원자번호 > 조직밀도 > 조영증강도

📚 출처: 1장_CT의_기초이론_복사본.pdf, p.46
Q9. 질량감약계수
물리
물, 얼음, 수증기의 질량감약계수에 대한 설명으로 옳은 것은?
  • ①물 > 얼음 > 수증기 순으로 크다
  • ②밀도가 큰 얼음이 가장 크다
  • ③선감약계수와 같은 값을 갖는다
  • ④세 물질 모두 동일한 값(0.214)을 갖는다
  • ⑤수증기가 가장 큰 값을 갖는다
해설
정답④

질량감약계수(μ/ρ)는 물리적 상태와 무관하게 동일 물질이면 항상 일정하다. 물·얼음·수증기는 모두 H₂O이므로 질량감약계수는 0.214 cm²/g로 동일하다.

단, 선감약계수는 밀도에 따라 달라 물(0.214) > 얼음(0.196) > 수증기(0.000128)로 모두 다르다.

📚 출처: 1장_CT의_기초이론_복사본.pdf, p.40–41
Q10. Gray scale
수치 암기
CT 장치와 모니터의 gray scale에 대한 설명으로 옳은 것은?
  • ①CT 장치는 8 bit, 256 단계이다
  • ②CT 장치는 12 bit, 4,096 단계(−1,024 ~ +3,071)이다
  • ③모니터는 16 bit, 65,536 단계이다
  • ④CT 장치와 모니터의 단계 수는 동일하다
  • ⑤CT 장치는 1,000 단계, 모니터는 500 단계이다
해설
정답②

CT 장치: 12 bit, 총 4,096 단계 (−1,024 ~ +3,071)

모니터(전시장치): 8 bit, 총 256 단계 (0 ~ 255)

이 차이 때문에 Windowing 기법으로 관심 영역만 집중적으로 표시한다.

Q11. Window Width
빈출 ★★
Window Width를 좁게(낮게) 설정하였을 때 나타나는 변화로 옳은 것은?
  • ①Gray scale 증가, contrast 저하
  • ②Contrast 저하, resolution 저하
  • ③Contrast 증가, resolution 증가, 비슷한 조직 식별능 증가
  • ④흑색과 백색이 약해지고 gray scale 증가
  • ⑤비슷한 조직 식별능이 저하된다
해설
정답③

Window Width를 좁게 설정하면: Gray scale 감소 / Contrast 증가 / 흑색·백색 강해짐 / Resolution 증가 / 비슷한 조직 식별능 증가.

반대로 WW를 넓게 설정하면 gray scale은 증가하지만 contrast와 resolution은 저하된다.

📚 출처: 1장_CT의_기초이론_복사본.pdf, p.74–75
Q12. Window Level
빈출 ★
Window Level을 높게 설정하면 상대적으로 잘 관찰되는 조직은?
  • ①공기
  • ②지방
  • ③폐 실질
  • ④뼈
  • ⑤물
해설
정답④

Window Level을 높이면 영상이 어두워지면서, X선 흡수도가 높은(선감약계수가 큰) 조직 — 즉 뼈(bone)와 같은 조직이 잘 관찰된다.

반대로 WL을 낮추면 영상이 밝아지며 공기·지방처럼 흡수도가 낮은 조직이 잘 관찰된다.

Q13. Window Setting
임상 ★
일반적으로 사용되는 Lung window setting 값은?
  • ①WW 80~100 / WL 0~30
  • ②WW 1,000~1,500 / WL −700 ~ −1,000
  • ③WW 400~500 / WL 0~100
  • ④WW 1,000~2,000 / WL 200~300
  • ⑤WW 300~350 / WL 0~30
해설
정답②

Lung window: WW 1,000~1,500 / WL −700 ~ −1,000. 폐 실질은 공기를 많이 포함하므로 WL이 매우 낮은 음수값이어야 한다.

① 뇌 실질 / ③ 흉부 mediastinal / ④ Bone / ⑤ 복부 연부조직

Q14. 데이터 획득 방식
개념
Axial scan(conventional scan)에 대한 설명으로 옳은 것은?
  • ①나선형 스캔을 통한 체적 데이터 획득 방식이다
  • ②동일 위치에서 임의의 시간 간격으로 연속 스캔하는 방식이다
  • ③테이블이 움직이는 동안 연속적으로 스캔한다
  • ④IGD(Inter Group Delay)가 적용된다
  • ⑤테이블 고정 상태에서 X선관이 회전하며 한 단면을 획득하는 step and shoot 방식이다
해설
정답⑤

Axial scan(conventional scan)은 테이블이 고정된 상태에서 X선관이 회전하며 한 단면의 데이터를 획득하는 방식으로, step and shoot CT technology로 불리며 슬라이스마다 ISD(Inter Scan Delay)가 존재한다.

① Volume scan / ② Cine scan / ③ Dynamic scan / ④ Cluster scan

Q15. Scan data vs Image data
개념
CT의 Scan data(raw data)와 Image data에 관한 설명으로 옳지 않은 것은?
  • ①Scan data는 sinogram 형태를 가진다
  • ②Image data의 양이 Scan data의 양보다 크다
  • ③Image data는 CT number로 표현된 공간 영역 데이터이다
  • ④Scan data는 재구성(back projection/convolution) 처리가 되지 않은 상태의 데이터이다
  • ⑤FBP(Filtered Back Projection)를 통해 scan data가 image data로 변환된다
해설
정답②

Raw data(scan data)의 양 > Image data의 양이다. 재구성 과정에서 필요 없는 데이터가 삭제되기 때문이다.

Scan data = measurement data = sinogram(주파수 영역) / Image data = CT number로 표현된 공간 영역 데이터.
