Q 01
3세대 CT 장치에 대한 특징으로 옳은 것은?
정답 ② 단일 연필형 선속
관련 이론 — CT 세대별 특징
세대엑스선 선속검출기움직임특징
1세대연필형 선속(pencil beam)1~2개Translate–Rotate1회 스캔 4~5분, 매우 느림
2세대좁은 부채꼴(narrow fan)3~52개Translate–Rotate1세대보다 빠름, 여전히 병진운동 존재
3세대넓은 부채꼴(wide fan beam)곡선배열 300~1000개Rotate–Rotate튜브+검출기 동시회전, 링 아티팩트 발생
4세대부채꼴(fan beam)고정식 원형배열 600~4800개Rotate only(튜브만)검출기는 고정, 링 아티팩트 적음
5세대전자빔(EBCT)고정움직임 없음심장 촬영용, 50ms 초고속
보기 분석
  • ①전자총 — 5세대 EBCT 특징
  • ②단일 연필형 선속 — ✗ 1세대 특징 (3세대는 넓은 부채꼴)
  • ③원형 인공물 — 3세대 특징 ✓ (실제 정답)
  • ④고정양극 사용 — 구형 CT에서만, 세대 구분 핵심 아님
  • ⑤고정식 검출기 — 4세대 특징

※ 3세대의 핵심 특징은 넓은 부채꼴 선속 + 튜브와 검출기 동시회전 + 링 아티팩트(원형 인공물)입니다. 정답은 ③ 원형 인공물입니다.

✦ 암기 포인트

1세대 연필 + 1~2개 검출기 → 가장 느림

2세대 좁은 부채 + 병진회전

3세대 넓은 부채 + 동시회전 → Ring artifact

4세대 검출기 고정(원형) + 튜브만 회전

5세대 전자빔(EBCT), 심장

Q 02
Window width와 level에 대한 설명으로 옳은 것은?
정답 ③ Window width가 감소하면 분해능이 감소한다 → 실제 정답: ② Window level이 증가하면 영상이 밝게 나타난다
관련 이론 — Window 설정

Window Level (WL, 창높이): 영상에서 표시할 HU 범위의 중심값. 밝기를 조절.

Window Width (WW, 창폭): 표시할 HU 범위의 폭. 대조도를 조절.

조작결과
WL 증가 ↑영상이 어두워짐 (기준점이 높아져 대부분 조직이 아래쪽에 위치)
WL 감소 ↓영상이 밝아짐
WW 증가 ↑대조도 감소, 넓은 범위 표시 (뼈 관찰)
WW 감소 ↓대조도 증가, 좁은 범위만 표시 (연부조직 구별)

※ Window는 디스플레이 조작일 뿐, 실제 공간분해능(해상도)과는 무관합니다.

보기 분석
  • ①WL 감소 → 뼈가 잘 나타난다 — ✗ WL 감소 시 영상이 밝아져 연부조직이 잘 보임
  • ②WL 증가 → 영상이 밝게 — ✗ WL 증가 시 어두워짐 (반대 설명)
  • ③WW 감소 → 분해능 감소 — ✗ WW는 대조도와 관련, 분해능과 무관
  • ④WW 증가 → 대조도 감소 — ✓ 정답
  • ⑤WW·WL 증가 → 노이즈 증가 — ✗ 디스플레이 설정은 노이즈 영향 없음

※ 정답은 ④ Window width가 증가하면 대조도가 감소한다입니다.

✦ 암기 포인트 — Window 4방향

WL ↑ 어두움 / WL ↓ 밝음 (밝기 조절)

WW ↑ 대조도 ↓ / WW ↓ 대조도 ↑ (대조도 조절)

외우는 법: "Width는 넓어질수록 뭉쳐 보인다(대조도↓)" / "Level은 올라가면 기준이 높아져 아래(어두운)로 내려간다"

Q 03
CT검사 시 전처리 과정 후의 데이터의 명칭으로 옳은 것은?
정답 ③ 투영 데이터(projection data)
관련 이론 — CT 데이터 흐름

CT 영상 형성 과정에서 데이터는 여러 단계로 변환됩니다.

단계명칭설명
1측정 데이터 (measurement data)검출기가 받은 원시 신호(아날로그)
2원 데이터 (raw data)A/D 변환된 디지털 데이터, 아직 보정 안 됨
3투영 데이터 (projection data)전처리(보정) 완료 → 재구성 직전 단계
4필터 된 데이터 (filtered raw data)컨볼루션 필터 적용 후
5이미지 데이터 (image data)역투영 완료 → 최종 영상

전처리 = 검출기 감도 보정, 빔 하드닝 보정 등. 전처리 후 재구성 입력으로 쓰이는 것이 projection data입니다.

보기 분석
  • ①원 데이터(raw data) — ✗ 전처리 이전 단계
  • ②이미지 데이터 — 재구성 완료 후 최종 영상
  • ③투영 데이터 — ✓ 정답 (전처리 직후)
  • ④측정 데이터 — 검출기 원시 신호, 가장 초기
  • ⑤필터 된 데이터 — projection 데이터에 컨볼루션 필터 적용 후
✦ 암기 포인트 — 데이터 순서

측정 → 원(raw) → [전처리] → 투영(projection) → 필터링 → 이미지

"측원투필이" 순서로 외우기

Q 04
다음 영상에 대한 설명으로 옳은 것은?
이미지 설명: 좌측은 시간-HU 그래프 (시간 축에 HU값 변화 곡선, "average 228.0"), 우측은 흉부 CT 횡단면 영상에서 ROI가 하행대동맥(descending aorta)에 설정된 모습. 즉 Bolus tracking(볼러스 트래킹) 영상입니다.
정답 ② 시간변화에 따른 조영제의 주입량을 알 수 있다 → 실제 정답: ② 시간변화에 따른 조영제의 농도변화를 알 수 있다 계열 / ⑤ ROI가 200HU에 도달할 때 스캔
관련 이론 — Bolus Tracking

Bolus Tracking: 조영제 주입 후 목표 혈관의 HU값이 임계값(threshold, 보통 100~200HU)에 도달하는 순간을 자동 감지해 본 스캔을 시작하는 기법.

  • 관심영역(ROI)을 대동맥(ascending/descending aorta)에 설정
  • 저선량 모니터링 스캔을 반복하며 HU값 변화 추적
  • 임계값 도달 → 자동으로 본 스캔 트리거
  • 목적: 최적 동맥기(arterial phase) 포착, 환자 간 순환시간 차이 보정

영상의 ROI 위치(하행대동맥, 척추 앞쪽의 둥근 구조)로 보아 흉부/복부 CT 조영검사의 bolus tracking입니다.

보기 분석
  • ①관심영역은 오름대동맥 — ✗ 영상은 하행(내림)대동맥, 횡단면 위치로 구분
  • ②시간변화에 따른 조영제 주입량 — ✗ 주입량이 아니라 HU(농도)변화를 측정
  • ③동맥조영 효과 극대화 검사 — ✓ 정답 후보
  • ④혈관외유출 예방 — ✗ 이건 extravasation 모니터링, 다른 기법
  • ⑤ROI 내가 200HU 도달 시 높은 선량 스캔 — ✓ 정답 후보 (임계값 도달 시 본 스캔)

※ 가장 정확한 정답은 ⑤ ROI 내 HU값이 임계값(예: 200HU)에 도달할 때 본 스캔(고선량)을 시행한다입니다. Bolus tracking의 핵심 작동 원리를 정확히 설명합니다.

✦ 암기 포인트 — Bolus Tracking

ROI는 대동맥에 설정 → 저선량 모니터링

HU값이 임계값(100~200HU) 도달 → 본 스캔 자동 시작

목적: 동맥기 최적 포착, 개인차 보정

유사 기법: Test Bolus(소량 조영제로 시간 측정 후 본 스캔)

Q 05
다음 중 CT number가 가장 낮은 것은?
정답 ③ 지방
관련 이론 — CT number (HU) 기준값

CT number는 물을 기준(0 HU)으로 물질의 선감약계수를 상대적으로 나타낸 값입니다.

조직HU 값
공기-1000
폐-500 ~ -800
지방-100 ~ -50
물0
뇌척수액0 ~ +15
근육+40 ~ +50
이자(췌장)+30 ~ +50
혈액(응고)+50 ~ +80
뼈+400 ~ +1000

보기 중에서는 지방(약 -100HU)이 가장 낮습니다. 공기/폐가 있다면 더 낮지만 보기에 없음.

보기 분석
  • ①물 — 0 HU
  • ②근육 — +40 ~ +50
  • ③지방 — ✓ -100 ~ -50 (가장 낮음)
  • ④이자 — +30 ~ +50
  • ⑤뇌척수액 — 0 ~ +15
✦ 암기 포인트 — HU 순서

공기(-1000) < 폐 < 지방(-100) < 물(0) < CSF < 근육·장기(+40~50) < 혈액 < 뼈(+1000)

"공폐지 물뇌근혈뼈" — 음수에서 양수로

Q 06
다음 중 부분체적 현상에 의한 인공물을 감소시키는 방법으로 옳은 것은?
정답 ② 단면두께 감소
관련 이론 — 부분체적 효과(Partial Volume Effect)

부분체적 효과(PVE): 한 복셀(voxel) 안에 여러 다른 물질(예: 뼈+뇌조직)이 섞여 있을 때, 그 복셀의 HU값이 평균값으로 나타나 경계가 흐릿해지고 실제와 다른 값이 표시되는 현상.

  • 원인: 복셀(voxel) 크기가 조직 경계보다 클 때
  • 주로 발생 부위: 측두골, 후두와(posterior fossa), 폐결절 주변
  • 해결법: 슬라이스 두께(단면두께)를 얇게 → 복셀 z축 크기 감소 → 평균화 감소
보기 분석
  • ①검출기 교체 — ✗ 검출기는 PVE와 직접 관련 없음
  • ②단면두께 감소 — ✓ 정답
  • ③저 관전압 사용 — ✗ 빔 하드닝 아티팩트와 관련
  • ④높은 관전류 사용 — ✗ 관전류는 노이즈(SNR)와 관련
  • ⑤움직이지 않도록 고정 — ✗ 움직임 인공물 감소법
✦ 암기 포인트 — 인공물별 감소법

부분체적 → 슬라이스 두께 ↓

빔 하드닝 → 콜리메이터, 필터, 고관전압

움직임 → 고정, 호흡정지, 빠른 스캔

금속 → MAR 알고리즘, 고관전압

노이즈 → mAs ↑ (관전류 ↑)

Q 07
다음 중 CT 엑스선관의 구비조건으로 옳은 것은?
정답 ③ 열용적 용량이 커야 한다
관련 이론 — CT X-ray tube 구비조건

CT는 수초~수십 초 동안 고속 연속 촬영하므로 X-ray tube에 요구되는 조건이 일반 진단용과 다릅니다.

  • 초점은 작아야 한다 (분해능 향상) — 단, 너무 작으면 열 집중 문제
  • 고진공도 유지
  • 열용량(heat capacity)이 커야 한다 — 핵심! 수 MHU 이상 요구
  • 높은 방열률 (빠른 냉각)
  • X선 흡수가 적어야 한다 (tube housing)
  • 높은 관전류 발생 가능
  • 회전양극 사용 (고정양극 X)
보기 분석
  • ①초점이 커야 한다 — ✗ 반대, 작아야 분해능 ↑
  • ②저 진공도 유지 — ✗ 고진공도 유지
  • ③열용적 용량이 커야 한다 — ✓ 정답
  • ④X선 흡수가 커야 한다 — ✗ 반대, 적어야 함
  • ⑤낮은 관전류 발생 — ✗ 높은 관전류 필요
✦ 암기 포인트 — CT 튜브 요구사항

"작·고·커·높"

초점 작게 / 진공도 고 / 열용량 커야 / 관전류·방열률 높이

Q 08
나선형 CT 장치에서 고전압인가(high voltage slip ring) 방식의 특징으로 옳은 것은?
정답 ③ 높은 출력 제공
관련 이론 — Slip Ring 방식

Slip Ring: 나선형(helical) CT에서 튜브가 연속 회전할 수 있도록 케이블 대신 접촉 전극(브러시)을 사용해 전원을 공급하는 장치. 두 방식이 있습니다.

구분저전압 slip ring고전압 slip ring
전압수백 V (AC)수십~150 kV (DC)
고전압 발생기 위치갠트리 내부(회전체)갠트리 외부(고정)
출력상대적으로 높음낮음~중간
전기 누출·절연유리문제 가능
안정성신뢰성 ↑상대적으로 ↓
크기·무게회전부 무거움회전부 가벼움

※ 고전압 slip ring은 갠트리가 가벼워지는 장점이 있지만, 고전압 절연·누출·신뢰성 면에서 저전압 방식이 더 우수해 현대 CT는 저전압 slip ring이 주류입니다.

보기 분석
  • ①신뢰성 있는 작동 — ✗ 저전압 방식 특징
  • ②높은 회전 부하 — ✗ 오히려 회전부가 가벼움
  • ③높은 출력 제공 — △ 애매함, 엄밀히는 저전압이 출력 유리
  • ④전기절연 불필요 — ✗ 고전압이므로 절연이 매우 중요
  • ⑤전기적 누출 없음 — ✗ 오히려 누출 우려 ↑

※ 교재에 따라 고전압 방식의 "고출력 가능"을 장점으로 제시하기도 합니다. 단, 현장에서는 저전압 방식이 더 많이 쓰입니다. 정답은 ③ 높은 출력 제공으로 채점됩니다.

✦ 암기 포인트 — Slip Ring 비교

저전압: 발생기 내부, 회전부 무거움, 안정성·신뢰성 ↑ (현대 주류)

고전압: 발생기 외부, 회전부 가벼움, 절연·누출 문제

Q 09
다음 중 soft algorithm의 적용이 가장 적합한 부위로 옳은 것은?
정답 ⑤ 복부
관련 이론 — 재구성 알고리즘(Kernel)

CT 재구성 시 사용되는 컨볼루션 커널(filter)은 대상 조직에 따라 선택합니다.

알고리즘특성적용 부위
Soft (smooth)저대조도 ↑, 노이즈 ↓, 공간분해능 ↓복부, 간, 뇌조직, 연부조직
Standard중간 균형일반 복부·흉부
Bone / Sharp공간분해능 ↑, 노이즈 ↑뼈, 폐, 측두골, 내이, 부비동
Ultra high / Edge최고 분해능, 노이즈 매우 ↑내이 세부구조, 폐 HRCT

Soft algorithm의 핵심: 저대조도 해상력이 중요한 부위(간 종괴, 췌장 병변 등)에 사용. 복부가 대표적.

보기 분석
  • ①폐 — sharp/lung kernel 사용
  • ②척추 — bone kernel 사용
  • ③골반 — 부위별로 다름, 뼈는 bone kernel
  • ④관자뼈 — ✗ ultra-high/bone kernel (고분해능 필요)
  • ⑤복부 — ✓ 정답, soft algorithm 대표 부위
✦ 암기 포인트 — Kernel 선택

Soft: 복부·간·뇌 → 저대조도 강조

Bone/Sharp: 뼈·폐·측두골·부비동 → 고해상도 강조

Trade-off: 분해능 ↑ ↔ 노이즈 ↑ (둘 다 만족 불가)

관련 이론 — 축별 분해능 결정 인자
축의미결정 인자
X-Y축 (횡단면)면내 분해능FOV, 매트릭스 크기, 검출소자 크기, 초점 크기, 재구성 알고리즘

  • ④장폭과 창높이 — 디스플레이 설정
  • ⑤슬라이스 두께 — ✓ 정답

Q 12
다음 영상(A, B)에서 A와 B의 화질 차이 발생 요인으로 옳은 것은?
이미지 설명: 1번 영상(A)은 선명한 혈관 3D VR(volume rendering), 2번 영상(B)은 계단 모양(step artifact)이 뚜렷한 거친 혈관 3D 영상. 같은 부위의 대동맥·혈관 분기부를 다른 조건으로 재구성한 결과물로 보입니다.
정답 ④ 절편두께 차이
관련 이론 — 3D 재구성 화질 결정 요소

3D VR이나 MIP 영상에서 계단 모양(step/stair-step artifact)이 나타나는 원인은 주로 Z축 샘플링 문제입니다.

  • 슬라이스 두께(절편두께)가 두꺼우면 → Z축 복셀이 커짐 → 3D 표면이 계단처럼 거칠어짐
  • 두께를 얇게 하면 → 등방성(isotropic) 복셀에 가까워져 매끄러운 3D 영상 생성
  • 같은 원리로 재구성 간격(overlap)을 좁히면 화질 향상

영상 1번은 얇은 절편으로 재구성한 매끈한 VR, 영상 2번은 두꺼운 절편으로 인한 step artifact가 뚜렷한 VR입니다.

보기 분석
  • ①알고리즘 차이 — 영향 있지만 이 차이의 주 원인은 아님
  • ②창폭·창높이 차이 — 디스플레이 설정, 3D 거칠기와 무관
  • ③관전압·관전류 차이 — 노이즈·대조도에 영향, 3D 계단 모양 원인 아님
  • ④절편두께 차이 — ✓ 정답
  • ⑤조영제 유무 — 두 영상 모두 조영된 상태
✦ 암기 포인트 — 3D 영상 화질

3D 영상의 매끄러움 = 슬라이스 두께가 지배

두께 얇게 → 등방성 복셀 → step artifact ↓

MDCT가 3D에 유리한 이유: 얇은 절편 + 등방성 복셀 구현 가능

Q 13
다음 중 CT number에 대한 설명으로 옳은 것은?
정답 ② 선감약계수에 비례한다
관련 이론 — CT Number (HU, Hounsfield Unit)

CT number는 물질의 선감약계수(μ)를 물의 선감약계수를 기준으로 상대적으로 표현한 값입니다.

CT number (HU) = 1000 × (μ물질 − μ물) / μ물

  • 기준 물질은 물(0 HU), 공기는 -1000 HU
  • μ(선감약계수)에 비례: 감약 큰 물질 = HU 큰 값
  • μ는 X선 에너지(kVp)에 따라 변화 → CT number도 관전압에 따라 달라짐
  • 모니터 크기와는 무관 (디스플레이 픽셀 수와는 별개)
보기 분석
  • ①기준 물질은 지방 — ✗ 기준은 물(water)
  • ②선감약계수에 반비례 — ✗ 비례해야 함
  • ③공기의 CT number는 -100 — ✗ -1000 HU
  • ④모니터 크기에 따라 달라짐 — ✗ 무관
  • ⑤입사 에너지에 따라 달라짐 — ✓ 정답

※ 가장 정확한 답은 ⑤ CT number는 입사되는 에너지에 따라 다르게 나타난다입니다. X선 에너지(kVp)가 바뀌면 μ가 달라지므로 HU값도 달라집니다.

✦ 암기 포인트 — CT number 기본

기준: 물 = 0, 공기 = -1000, 뼈 ≥ +1000

μ에 비례, X선 에너지(kVp)에 따라 변화

단위: HU(Hounsfield Unit)

Q 14
window width가 100, window level이 0이라면 영상에 표시되는 CT number의 범위는?
정답 ③ -50 ~ +50
관련 이론 — Window 범위 계산

Window의 표시 범위는 다음 공식으로 계산합니다.

하한 = WL − (WW/2)
상한 = WL + (WW/2)

WW=100, WL=0일 때:

  • 하한 = 0 − 50 = -50 HU
  • 상한 = 0 + 50 = +50 HU
  • 범위: -50 ~ +50

이 범위 밖의 값들은:

  • +50 이상 → 모두 흰색(포화)
  • -50 이하 → 모두 검은색
  • -50 ~ +50 범위만 회색조(gray scale)로 표시
보기 분석
  • ①-100 ~ +100 — ✗ WW=200일 때
  • ②-100 ~ 0 — ✗ WL=-50, WW=100
  • ③-50 ~ +50 — ✓ 정답
  • ④0 ~ +100 — ✗ WL=50, WW=100
  • ⑤+50 ~ +150 — ✗ WL=100, WW=100
✦ 암기 포인트 — Window 계산 공식

[WL − WW/2] ~ [WL + WW/2]

WL = 중심, WW/2 = 반폭 → 중심 ± 반폭

예: 뇌(35/80) → -5 ~ 75 HU

예: 폐(-600/1500) → -1350 ~ +150 HU

Q 15
CT 영상 재구성 과정에서 알고리즘을 적용하는 단계로 옳은 것은?
정답 ④ 컨볼루션 단계
관련 이론 — CT 재구성 과정

필터링된 역투영(Filtered Back Projection, FBP)의 단계별 흐름:

  1. 전처리(preprocessing): raw data 보정
  2. 컨볼루션(convolution): 재구성 커널(알고리즘) 적용 — soft/standard/bone 등 선택
  3. 역투영(back projection): 필터링된 데이터를 역방향으로 투영해 2D 영상 생성
  4. 후처리(post-processing): MPR, 3D, MIP 등

※ Soft/Bone/Sharp 같은 재구성 알고리즘은 컨볼루션 단계에서 적용됩니다. 재구성 단계는 역투영까지를 포함하고, 알고리즘은 그 안의 "커널 적용" 단계를 지칭합니다.

보기 분석
  • ①전처리 단계 — ✗ 데이터 보정 단계
  • ②재구성 단계 — △ 큰 범주에서는 맞으나 정확한 단계는 컨볼루션
  • ③후처리 단계 — ✗ MPR·3D 단계
  • ④컨볼루션 단계 — ✓ 정답
  • ⑤3차원 영상화 단계 — ✗ 후처리
✦ 암기 포인트 — 재구성 4단계

전처리 → 컨볼루션(커널=알고리즘) → 역투영 → 후처리

"전컨역후" — 알고리즘은 컨볼루션에서!

Q 16
CT 장치에서 다색 빔 사용으로 빔 경화현상이 발생하기 쉬운데 이를 보정하기 위해 사용되는 부속 기기로 옳은 것은?
정답 ⑤ 전압분배장치 → 실제 정답: ④ 나비형 필터(bowtie filter) 또는 ② 콜리메이터
관련 이론 — 빔 경화(Beam Hardening)

Beam Hardening: 다색 X선 빔이 물체를 투과할 때 저에너지 광자가 먼저 흡수되어 평균 에너지가 높아지는(빔이 "단단해지는") 현상. 주변부 HU값이 낮아지는 인공물을 만듭니다.

보정 방법/기기:

  • Bowtie filter(나비형 필터): X선관 출구에 장착, 주변부 광자를 더 많이 감쇄시켜 빔 균일화
  • 부가필터(Al, Cu): 저에너지 광자 제거로 빔 단색화
  • 고관전압 사용: 평균 에너지 높여 효과 감소
  • 소프트웨어 보정 알고리즘
보기 분석
  • ①슬립링 — 전원 공급 장치, 빔 보정과 무관
  • ②콜리메이터 — 빔 형태 제한 (산란선 감소 목적, 경화 보정은 아님)
  • ③나비형 필터(bowtie filter) — ✓ 정답
  • ④광전자증배관 — 검출기 구성 요소(일부 구형 방식)
  • ⑤전압분배장치 — ✗ 전원 계통 장치

※ 정답은 ③ 나비형 필터(bowtie filter)입니다. 빔 경화 보정의 대표 기기입니다.

✦ 암기 포인트 — Beam Hardening 대책

Bowtie filter (나비형) → 주변부 감쇄, 빔 균일화

부가필터 (Al, Cu) → 저에너지 제거

고kVp → 평균에너지 ↑

결과물로 나타나는 인공물: Cupping artifact(균일한 물체 중앙이 더 어둡게), streak artifact(뼈 사이 검은 줄)

Q 17
CT에서 매개변수 변경 시 노이즈가 감소하는 경우로 옳은 것은?
정답 ⑤ 산란선 증가 → 실제 정답: ④ 절편두께 증가
관련 이론 — CT 노이즈 결정 인자

CT에서 노이즈는 주로 양자 노이즈(quantum mottle)로, 검출기에 도달하는 광자 수와 관련됩니다. 광자 수가 많을수록 노이즈는 감소합니다(1/√N 법칙).

인자 변경노이즈 변화
관전압(kVp) ↑감소 (광자 에너지·수 증가)
관전류(mAs) ↑감소 (광자 수 증가)
절편두께 ↑감소 (복셀당 광자 ↑)
매트릭스 ↑ (픽셀 ↓)증가 (픽셀당 광자 ↓)
산란선 ↑증가
재구성 알고리즘(sharp)증가
보기 분석
  • ①관전압 감소 — ✗ 감소 시 노이즈 증가
  • ②관전류 감소 — ✗ 감소 시 노이즈 증가
  • ③산란선 증가 — ✗ 산란선은 노이즈 증가 요인
  • ④절편두께 증가 — ✓ 정답 (광자 수 증가)
  • ⑤매트릭스 증가 — ✗ 픽셀 크기 작아져 노이즈 증가

※ 정답은 ④ 절편두께 증가입니다. 단, 절편두께 증가는 Z축 해상도 감소라는 trade-off가 있음을 기억하세요.

✦ 암기 포인트 — 노이즈 감소법

광자 수 ↑ = 노이즈 ↓ (1/√N 법칙)

증가시켜야 할 것: mAs, kVp, 슬라이스 두께, 픽셀 크기(FOV)

증가하면 노이즈 증가하는 것: 매트릭스, 산란선, sharp kernel

Q 18
다음 중 공간분해능을 증가시키기 위한 방법으로 옳은 것은?
정답 ③ 단면두께를 얇게 한다
관련 이론 — 공간분해능 향상 방법

공간분해능(spatial resolution): 얼마나 작은 구조물 두 개를 구별할 수 있는가. 복셀/픽셀 크기를 줄이는 모든 방법이 유효합니다.

  • FOV ↓ (작게) → 픽셀 크기 ↓ → X-Y 분해능 ↑
  • 매트릭스 ↑ → 픽셀 수 ↑ → 픽셀 크기 ↓
  • 슬라이스 두께 ↓ → Z축 분해능 ↑
  • Sharp/Bone kernel 사용 → 경계 강조
  • 초점 크기 ↓ → 반그림자 감소
  • 검출소자 크기 ↓, 수 ↑

Trade-off: 공간분해능 ↑ ↔ 노이즈 ↑ / 선량 ↑

보기 분석
  • ①관전압 증가 — ✗ 노이즈·대조도 관련
  • ②관전류 증가 — ✗ 노이즈 감소와 관련
  • ③단면두께를 얇게 — ✓ 정답 (Z축 분해능 ↑)
  • ④연부조직 알고리즘 적용 — ✗ soft kernel은 분해능 낮음
  • ⑤SFOV·DFOV를 크게 — ✗ 반대, 작게 해야 픽셀 크기 ↓
✦ 암기 포인트 — 공간분해능 ↑

"복셀을 작게" 만드는 모든 것

FOV ↓ · 매트릭스 ↑ · 두께 ↓ · sharp 커널 · 초점 ↓

대가: 노이즈 ↑, 선량 ↑

Q 19
CT 촬영 시 pitch를 증가시킬 때 나타나는 현상으로 옳은 것은?
정답 ③ 검사시간 단축
관련 이론 — Pitch

Pitch = (테이블 이동거리 / 회전) ÷ 슬라이스 두께(또는 beam width)

Pitch 증가 (↑)영향
테이블 이동 빨라짐검사 시간 ↓
단위시간당 조사량 ↓피폭선량 ↓

  • ①A는 창폭(WW)이 넓다 — ✗ A가 좁고, B가 넓음
  • ②A는 폐 실질 관찰 window — ✗ A는 종격동 window
  • ③B는 창높이(WL)가 높다 — ✗ B의 WL은 -600으로 낮음
  • ④B는 high resolution algorithm 사용 — ✓ 정답
  • ⑤B는 외상에 의한 골절 의심 시 용이 — ✗ 골절은 bone window
✦ 암기 포인트 — 흉부 CT 두 창

Q 29
CT에서 HU 값에 영향을 주는 인자로 옳은 것은?
정답 ④ 방사선속의 에너지
관련 이론 — HU 값 결정 요인

HU는 선감약계수(μ)에 의해 결정되며, μ는 X선 에너지에 의존합니다.

  • X선 에너지(kVp) ↑ → 조직의 μ 감소 → 뼈 HU 감소, 연부조직 HU 변화
  • 조영제의 HU는 kVp에 특히 민감 (요오드의 광전효과)
  • 빔 경화 정도(평균 에너지) → HU 변화 유발
  • 재구성 알고리즘(커널)은 HU 평균값에는 큰 영향 없지만 가장자리 HU는 변동 가능

스캔 방식, 검출기 수, 단면 수, 입력전원 공급방식은 HU 값 자체에 직접적인 영향을 주지 않습니다.

보기 분석
  • ①스캔 방식 — ✗ axial/helical은 HU 무관
  • ②검출기 수 — ✗ 분해능엔 영향, HU는 아님
  • ③스캔 단면의 수 — ✗ 해상도 관련
  • ④방사선속의 에너지 — ✓ 정답
  • ⑤입력전원의 공급방식 — ✗ 장비 동작 방식
✦ 암기 포인트 — HU 변동 요인

HU 값 = 조직 μ와 X선 에너지의 함수

주요 영향: kVp, 빔 경화 정도, 조영제 유무

영향 없음: 스캔 방식, 검출기 수, 전원

Q 30
CT 검사 시 환자 피폭선량 증가의 직접적인 요인으로 옳은 것은?
정답 ③ 높은 관전류
관련 이론 — CT 피폭선량 인자
변수피폭선량 영향
관전류(mAs) ↑선량 선형 비례 증가 (가장 직접적)
관전압(kVp) ↑선량 지수적 증가 (kVp² ~ kVp⁵)
스캔 시간 ↑선량 ↑
스캔 범위 ↑선량 ↑
Pitch ↑선량 감소
Pitch ↓선량 증가 (중첩 스캔)
AEC 사용선량 감소 (자동 조절)
반복 스캔선량 ↑

보기 중 직접적으로 선량을 증가시키는 것은 높은 관전류(mAs)입니다. 관전류는 광자 수에 선형 비례하므로 그대로 선량에 반영됩니다.

보기 분석
  • ①높은 pitch — ✗ pitch 높으면 선량 감소
  • ②낮은 관전압 — ✗ 저kVp는 선량 감소
  • ③높은 관전류 — ✓ 정답 (mAs ∝ 선량)
  • ④짧은 검사시간 — ✗ 시간 짧으면 선량 감소
  • ⑤AEC 사용 — ✗ AEC는 선량 최적화(감소) 기법
✦ 암기 포인트 — 선량 조절

선량 증가: mAs ↑, kVp ↑, 시간 ↑, pitch ↓, 스캔범위 ↑

선량 감소: AEC, 반복 피함, pitch ↑, 저kVp(소아)

지수 CTDI·DLP 기억: DLP = CTDIvol × 스캔길이
